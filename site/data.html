Open Data Access

All ratings data is freely available as JSON. No API key required. Updated daily by the StablecoinRatings pipeline.

Full Ratings
/data/ratings_full.json
Complete ratings with all pillar scores, sub-scores, sparklines, and metadata for every coin.

Grades Only

Compact coin → grade/score lookup. Ideal for algorithmic consumers. Includes mcap and rank.
Download JSON
Historical Archive

365-day rolling archive of all score snapshots. Each entry includes full pillar breakdown per coin per day.

Usage Example

// Fetch latest ratings fetch('https://stablecoinratings.org/data/grades.json') .then(r => r.json()) .then(data => { for (const [symbol, info] of Object.entries(data.coins)) { console.log(`${symbol}: ${info.grade} (${info.score})`); } }); // Historical score for a specific coin fetch('https://stablecoinratings.org/data/history.json') .then(r => r.json()) .then(hist => { const latest = hist.versions[hist.versions.length - 1]; console.log('USDT peg score:', latest.coins.USDT.peg); });
